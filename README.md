# 🔥 M_tabs - Interactive Tabs Component

## 📌 Description
M_tabs est un composant d’interface utilisateur moderne permettant de gérer des onglets interactifs (tabs) en JavaScript pur.

Ce projet démontre mes compétences en :
- HTML5
- CSS3
- JavaScript (DOM manipulation)

## 🚀 Démo en ligne
👉 https://meresathindehou-prog.github.io/M_tabs/

## 🎯 Fonctionnalités
- Navigation fluide entre onglets
- Interface simple et intuitive
- Code léger et performant

## 🛠️ Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Structure du projet

M_tabs/
│── index.html
│── style.css
│── script.js

## 💡 Cas d’utilisation
Ce composant peut être utilisé pour :
- Tableaux de bord (dashboard)
- Sites vitrines
- Applications web interactives
